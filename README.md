# Goal Predictor Bot

بوت توقع الأهداف مع نظام دفع مزدوج (باينانس + بنكي)

## المتطلبات

- Node.js 18 أو أعلى
- حساب Firebase
- بوت تليجرام
- قناة تليجرام

## الإعداد

1. انسخ الملفات إلى مستودع Git
2. انشئ خدمة جديدة على Render
3. اضبط متغيرات البيئة في لوحة تحكم Render
4. شغل البوت

## متغيرات البيئة المطلوبة

- `BOT_TOKEN`: توكن البوت
- `ADMIN_ID`: آيدي الأدمن
- `CHANNEL_ID`: آيدي القناة
- `CHANNEL_USERNAME`: يوزر القناة
- `FIREBASE_*`: إعدادات Firebase
- `PAYMENT_*`: روابط الدفع
